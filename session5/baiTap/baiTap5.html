5. Groups of selectors

A comma-separated list of selectors represents the union of all elements selected by each of the individual
selectors in the list. (A comma is U +002C). For example, in CSS when several selectior share the same
declarations, they may be grouped into a comma-sparated list. White space may appear before after the comma.

CSS example

In the example, we conlose three rules with identical delaration into one. thus

h1 { font-family: sans-serif }

h2 { font-family: sanns-serif }

h2 { font-family: sanns-serif }

is equivalent to:

h1, h2, h3, {font-family: sans-serif }
